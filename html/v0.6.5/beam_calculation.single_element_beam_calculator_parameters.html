beam_calculation.single_element_beam_calculator_parameters module

We define a base class to hold the parameters for the BeamCalculator.

SingleElementCalculatorParameters is attribute of an Element and holds parameters that depend both on the Element under study and the on the BeamCalculator solver that is used.

Currently, it is used by Envelope1D only, as TraceWin handles it itself.
